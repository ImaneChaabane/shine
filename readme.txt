Les plugins utilisés:

Elementor - Header, Footer & Blocks: Ce plugin permet de créer un en-tête et un pied de page personnalisés avec Elementor et de les afficher sur des emplacements sélectionnés. On peut également créer des blocs Elementor personnalisés et les placer n'importe où sur le site Web avec un shortcode.

Simply Static: Un générateur de site statique pour créer des versions statiques rapides et sécurisées du site WordPress.

Starter Templates: Modèles de démarrage est une solution complète pour les sites de démarrage complets, les modèles de page unique, les blocs et les images.

WooCommerce:Une boîte à outils de commerce électronique qui  aide à vendre quoi que ce soit.

WPForms Lite: Plugin de formulaire de contact WordPress
